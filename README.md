# Statistical-test-using-non-parametric-statistical-methods-using-R
A comprehensive collection of non-parametric statistical tests implemented in R, covering sign tests, Wilcoxon, Mann-Whitney, and Chi-Squared analysis for various research scenarios.
### Statistical Methods Implemented

#### 1. Sign Test
* **One-Sample Sign Test**: Tests the median of a single group against a theoretical value using the Binomial distribution.
* **Paired-Sample Sign Test**: Analyzes the direction of difference (positive/negative) between two related samples.

#### 2. Wilcoxon Signed-Rank Test
* Used as a more powerful alternative to the Sign Test because it considers both the direction and the **magnitude (rank)** of the differences.
* Applied to **Single Samples** and **Paired Samples** (e.g., comparing psychotic vs. control groups).

#### 3. Mann-Whitney U Test (Wilcoxon Rank-Sum Test)
* Used to determine whether there is a significant difference between two **independent groups** (e.g., experimental group vs. control group).

#### 4. Chi-Squared Test ($\chi^2$)
* **Test of Independence**: Evaluates the relationship or association between two categorical variables using contingency tables.
* **Test of Homogeneity**: Examines whether the frequency distribution of a specific characteristic is consistent across different populations.
